2019 Social Accounting Matrix for Kenya
Country: Kenya
Year: 2019
Units: Billions of Kenyan Shilling (Sh)

Citation requirement: The users of 2019 Social Accounting Matrix for Kenya Dataset are required to cite both the dataset and the associated data paper. Please see below for the recommended citation.

Dataset: International Food Policy Research Institute (IFPRI). 2021. 2019 Social Accounting Matrix for Kenya. Washington DC: IFPRI [dataset]. https://doi.org/10.7910/DVN/ALUXSI. Harvard Dataverse. Version 1.

Data Paper: Thurlow, James. 2021. 2019 Social Accounting Matrix for Kenya: A Nexus Project SAM. Data Paper. Washington, DC: International Food Policy Research Institute (IFPRI). doi: 10.2499/p15738coll2.134819. https://doi.org/10.2499/p15738coll2.134819.

Data files in this dataset:
1. IFPRI_SAM_KEN_2019_SAM - This file contains the main data table of social accounting matrix (SAM).
2. IFPRI_SAM_KEN_2019_POP - This file contains information on "Representative Household Populations" measured in 1000s of people.
